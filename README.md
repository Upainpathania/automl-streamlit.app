# 🚀 AutoML Web Application
LINK -https://automl-appapp-9f2ud68rz3ditlctgbe9gg.streamlit.app/ (CLICK TO USE THE APP

A powerful and interactive **Automated Machine Learning (AutoML)** web app that enables users to perform data analysis, visualization, and model building — all in one place, without complex coding.

## ✨ Key Features

### 📂 Data Upload & Exploration

* Upload your dataset (CSV format)
* Automatic dataset analysis:

  * Shape of dataset
  * Missing values detection
  * Duplicate records check
  * Data preview (Head & Tail)

### 📊 Data Visualization

* User-friendly graph selection
* Choose specific columns for visualization
* Generate meaningful insights through plots

### 🤖 Model Training & Customization

* Select:

  * Target column
  * Training data size (train-test split)
  * Machine learning model
  * Evaluation metrics
* Supports multiple ML algorithms

### ⚡ Model Comparison

* Train multiple models
* Compare performance based on selected metrics
* Identify the best-performing model easily

---

## 🧠 Workflow

1. Upload dataset
2. Explore dataset insights (shape, null values, duplicates, preview)
3. Visualize data using customizable graphs
4. Select model settings (target, split, metrics, algorithm)
5. Train models and compare results
6. Choose the best model for your use case

## 🛠 Tech Stack

* Python (Flask / FastAPI)

To simplify the machine learning pipeline by giving users full control over:
